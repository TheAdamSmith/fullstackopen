# Part 4: Bloglist Backend
The back end for managing blog lists and user login

Uses React, Express, and MongoDB. Jest is used for testing.


## Install

```bash
npm install
```

## Usage

Backend

Development
```bash
npm run dev
```
Production
```bash
npm start
```
Test (Jest)
```bash
npm run test
```
Test (Cypress)
```bash
npm run start:test
```
